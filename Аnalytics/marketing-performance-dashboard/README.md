# Marketing Performance Dashboard

## Опис проєкту

У цьому проєкті розроблено інтерактивний маркетинговий дашборд для аналізу результатів рекламних кампаній **Facebook Ads** і **Google Ads**.

Основна мета проєкту — об’єднати дані з різних рекламних джерел, підготувати їх до аналізу за допомогою SQL та створити у Tableau Public зручний дашборд для оцінювання ефективності кампаній.

Проєкт дає змогу аналізувати рекламні витрати, охоплення, покази, кліки, ліди, цінність конверсій та маркетингові показники в розрізі джерел, кампаній, груп оголошень і періодів.

## Джерела даних

У проєкті використовувалися дані рекламних кампаній із двох джерел:

- Facebook Ads;
- Google Ads.

Дані Facebook Ads були об’єднані з довідниками кампаній і груп оголошень. Після цього дані Facebook Ads і Google Ads були приведені до спільної структури та об’єднані в один набір для подальшого аналізу.

## Етапи виконання проєкту

### 1. Підготовка даних у SQL

За допомогою SQL було виконано:

- об’єднання таблиць Facebook Ads із довідниками кампаній і груп оголошень;
- приведення даних Facebook Ads і Google Ads до єдиної структури;
- об’єднання даних із різних рекламних платформ;
- очищення та декодування значень `utm_campaign`;
- обробку відсутніх значень за допомогою `COALESCE`;
- агрегацію рекламних показників;
- підготовку фінального набору даних для експорту в Tableau Public.

### 2. Розраховані показники

Для аналізу рекламної ефективності використовувалися такі показники:

- `spend` — рекламні витрати;
- `impressions` — кількість показів;
- `reach` — охоплення;
- `clicks` — кількість кліків;
- `leads` — кількість лідів;
- `value` — цінність отриманих результатів;
- CTR — показник клікабельності;
- CPC — вартість одного кліку;
- CPM — вартість тисячі показів;
- CPL — вартість одного ліда;
- ROMI — повернення маркетингових інвестицій.

### 3. Експорт даних

Результат SQL-запиту було експортовано у форматі CSV та використано як джерело даних для Tableau Public.

### 4. Побудова дашборда в Tableau Public

У Tableau Public було створено інтерактивний маркетинговий дашборд із ключовими показниками та графіками.

Основні елементи дашборда:

- KPI-картки;
- `Metric Trend` — динаміка вибраного показника;
- `Spend vs Leads Analysis` — співвідношення рекламних витрат і кількості лідів;
- `Campaign Ranking` — рейтинг рекламних кампаній;
- `Campaign Efficiency Analysis` — аналіз ефективності кампаній;
- параметр `Select Metric` для перемикання між маркетинговими показниками;
- інтерактивний `Dashboard Filter Action`.

## Інтерактивність дашборда

У дашборді реалізовано параметр `Select Metric`, який дає змогу перемикатися між різними маркетинговими показниками без створення окремого графіка для кожної метрики.

Також налаштовано `Filter Action`: після натискання на кампанію в графіку `Campaign Efficiency Analysis` інші візуалізації автоматично фільтруються за вибраною кампанією.

## Фільтри

Для детальнішого аналізу в дашборді передбачено фільтри:

- період дат;
- `source`;
- `campaign_name`;
- `adset_name` або `audience_segment`;
- `utm_campaign`.

## Основні можливості дашборда

Дашборд дає змогу:

- відстежувати динаміку маркетингових показників;
- порівнювати ефективність Facebook Ads і Google Ads;
- знаходити кампанії з найкращими та найгіршими результатами;
- аналізувати співвідношення витрат і кількості лідів;
- оцінювати вартість залучення користувачів;
- досліджувати ефективність окремих кампаній;
- швидко перемикатися між ключовими маркетинговими метриками;
- фільтрувати результати за джерелом, кампанією та періодом.

## Навички, продемонстровані в проєкті

- написання SQL-запитів;
- використання `JOIN`;
- об’єднання даних за допомогою `UNION ALL`;
- використання `CTE`;
- очищення текстових даних;
- декодування `utm_campaign`;
- обробка значень `NULL` за допомогою `COALESCE`;
- агрегація маркетингових даних;
- розрахунок маркетингових метрик;
- експорт даних у CSV;
- створення KPI-карток;
- побудова інтерактивних візуалізацій;
- робота з параметрами Tableau;
- налаштування фільтрів;
- створення Dashboard Actions;
- публікація проєкту в Tableau Public.

## Як переглянути проєкт

1. Відкрийте посилання на Tableau Public у розділі **«Посилання на дашборд»**.
2. Використовуйте фільтри для вибору періоду, джерела або рекламної кампанії.
3. Змініть параметр `Select Metric`, щоб переглянути різні маркетингові показники.
4. Натисніть на кампанію в `Campaign Efficiency Analysis`, щоб відфільтрувати пов’язані графіки.
